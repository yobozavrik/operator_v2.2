# UI Terminology

## Purpose
Єдиний словник термінів для ERP / Production Hub.

Цей файл потрібен, щоб:
- не змішувати українську, російську та англійську в UI
- не називати одну й ту саму сутність різними словами
- утримувати консистентність у всіх модулях

## Language Rule
- Увесь інтерфейс ERP — **українською мовою**.
- Англійські назви допустимі лише для:
  - брендів
  - усталених технічних скорочень, якщо вони реально потрібні
- Якщо український відповідник є зрозумілим і природним — використовувати саме його.

## Preferred Phrasing Patterns
### Good
- **Що запускати і де дефіцит**
- **Що потрібно бачити одразу**
- **Треба виготовити**
- **Відкрити деталі та обрати позиції**
- **Критичні відхилення**
- **Навантаження мережі**

### Avoid
- змішані конструкції на кшталт `Fill rate`, `Workshops`, `live-залишки`
- різні слова для одного сенсу (`огляд`, `overview`, `dashboard`, `обзор`) без правила
- російські підписи в українському UI

## Special Rule for Graviton
У Graviton зберігається щільний картковий операційний патерн.
Там особливо важливо, щоб терміни були:
- короткі
- скановані
- однозначні
- однакові у картках, модалках, деталізації та заявці
