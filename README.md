# Operator (Galia Baluvana) — Система управління виробництвом та логістикою

Система призначена для моніторингу залишків у реальному часі, планування виробництва та розподілу продукції між магазинами мережі "Galia Baluvana".

## 🚀 Основні функції
- **Live Monitoring**: Візуалізація залишків на основі Poster API.
- **Production Planner**: Моделювання виробничого плану на декілька днів.
- **Distribution Manager**: Автоматизований розподіл товару між локаціями.

## 🛠 Технології
- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS, SWR.
- **Backend**: Next.js API Routes.
- **Data**: Supabase (PostgreSQL), Poster API integration.

## 🏁 Початок роботи

### 1. Встановлення залежностей
```bash
npm install
```

### 2. Налаштування Environment Variables
Створіть файл `.env.local` на основі прикладу та додайте ключі:
- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`
- `POSTER_TOKEN`

### 3. Локальний запуск
```bash
npm run dev
```
Додаток буде доступний за адресою [http://localhost:3000](http://localhost:3000).

## ⚠️ Важливо
Проект використовує live-merge даних. Це означає, що дані про залишки в інтерфейсі беруться безпосередньо з Poster API та об'єднуються зі статистикою з бази даних без проміжного запису залишків у БД.
